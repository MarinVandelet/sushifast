# SushiFast

## Application web moderne permettant d’explorer différents menus de sushis, avec recherche en temps réel, animations fluides et interface premium.

Lien du site : (SushiFast)[https://sushifast.vercel.app]

##fonctionnalités principales

- Recherche en temps réel des menus (avec animation & icône search).
- Pages dédiées : Menus, Saveurs, Avocat/Coriandre, Sans California, Prix Total, Extrêmes.
- Animations Framer Motion sur la navigation, les cards et l’interface.
- Background animé (Canvas) en thème japonais minimaliste.
- UI moderne & responsive grâce à TailwindCSS.
- Page de détails dynamique pour chaque menu.

## Technologies utilisées

React
Vite
React Router DOM
TailwindCSS
Framer Motion 
Canvas API
JavaScript

## Installation
git clone https://github.com/ton-projet/sushifast.git
cd sushifast
npm install
npm run dev
